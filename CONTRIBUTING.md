# 🤝 Contributing to Claude Usage Monitor

Thanks for helping improve `claude-usage-monitor`.

---

## 🐞 Bugs and feature requests

Please open a GitHub issue for:

- bugs or regressions
- UI problems or Windows-specific quirks
- feature ideas or polish requests
- documentation fixes

Include screenshots, Windows version, and reproduction steps when possible.

---

## 🔀 Pull requests

PRs are welcome.

Please:

1. open an issue first for larger changes
2. keep PRs focused and easy to review
3. update docs when behavior changes
4. avoid committing local-only files such as `CLAUDE.md`, `NOTES.md`, build outputs, or personal config

---

## 👀 Review flow

This repo uses code ownership so **`@Godimas101`** is automatically requested for review on pull requests.
